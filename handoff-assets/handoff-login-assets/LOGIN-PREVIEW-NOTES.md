# Login preview — SVG & how to wire the colours

Файл: [`login-preview.svg`](login-preview.svg) — точний макет з Figma (262×164), із
захардкодженими дефолтними кольорами. Це прев'ю сторінки логіну в секції
**Appearance** вкладки **Login**.

Робоча версія (з CSS-змінними замість хардкоду) лежить інлайн у `login.html` —
шукай `class="tb-login-preview"`.

## Що зображає SVG

Це **прев'ю сторінки логіну**: сіра «сторінка» (фон), на якій по центру лежить
картка форми входу (аватар + заголовок, 2 поля вводу, 2 рядки «іконка+текст»,
кнопка).

## Мапа кольорів (4 керовані значення + статика)

Кожен `fill` у SVG відповідає одному контролу секеції Appearance. Щоб прев'ю
перемальовувалось наживо — SVG інлайниться в HTML, а `fill` замінюється на
`var(--…)` (зовнішній `<img src>` так не вміє, змінні крізь нього не проходять).

| Елемент у SVG | Дефолт | Керується | CSS-змінна |
|---|---|---|---|
| 2 великі path-и фону (весь прямокутник) | `#F0F0F0` / `#EEEEEE` | **Page background color** | `--login-page-bg` |
| Центральна картка форми | `#00695C` | **Primary palette** | `--login-primary` |
| Кнопка (низ картки) | `#FF5722` | **Accent palette** | `--login-accent` |
| 2 рядки «кружок + смужка» (`white`, opacity `0.85`/`0.55`) | білий | **Dark foreground** (чекбокс) | `--login-fg` |
| Аватар-кружок + заголовок-пілюля | `white` | — статика | — |
| 2 поля вводу | `#E8F4F2` | — статика (світлий tint) | — |

### Dark foreground
Чекбокс перемикає **лише 2 рядки «кружок+смужка»** (`--login-fg`):
- вимкнено → `#ffffff` (opacity зберігається: 0.85 для кружків, 0.55 для смужок)
- увімкнено → `#212121`

Аватар, заголовок і поля вводу **не** змінюються.

### Page background color
Це фон **самого SVG** (сіра «сторінка» навколо картки), а НЕ декоративна
рамка-контейнер секції (`.tb-login-box`) — та статична.

## Як зробити динамічним (для інтеграції)

1. Інлайнути SVG у розмітку (не через `<img>`).
2. Замінити `fill`:
   - `#F0F0F0`, `#EEEEEE` → `var(--login-page-bg)`
   - `#00695C` → `var(--login-primary)`
   - `#FF5722` → `var(--login-accent)`
   - у 4 path-ах foreground (ті, що `fill="white" fill-opacity=…`): `white` → `var(--login-fg)` (opacity лишити)
3. Дефолти на `:root`:
   ```css
   :root {
     --login-page-bg: #eeeeee;
     --login-primary: #00695c;
     --login-accent:  #ff5722;
     --login-fg:      #ffffff;
   }
   ```
4. З контролів оновлювати відповідну змінну на кореневому елементі
   (`document.documentElement.style.setProperty('--login-primary', hex)` і т.д.).
   Dark foreground: `--login-fg` = `#212121` | `#ffffff`.
